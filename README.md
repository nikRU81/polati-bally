# ПОЛАТИ БАЛЛЫ: Интерактивный калькулятор

## Описание проекта

Интерактивный калькулятор для моделирования программы лояльности «ПОЛАТИ БАЛЛЫ». Позволяет рассчитать экономический эффект программы с учётом инфляции, индексации ставок и различных сценариев поведения сотрудников.

## Основные функции

1. **Настройка параметров программы**
   - Установка базовой зарплаты и процента бюджета
   - Настройка уровня инфляции (ЗП и цен)
   - Установка периода и коэффициента индексации ставок

2. **Управление заданиями**
   - Перечень всех заданий с баллами
   - Категоризация заданий
   - Периодичность получения (разово/ежегодно/по условию)

3. **Расчёт экономического эффекта**
   - Итоговая таблица по годам с проверками
   - Сравнение с бюджетом нарастающим итогом
   - Проверка покупательной способности

4. **Моделирование сценариев**
   - Сотрудник с 1-го года программы (5 лет)
   - Сотрудник, пришедший позже (3 года)
   - Кастомные сценарии

## Структура файлов

```
polati_bally_calculator/
├── README.md                    # Описание проекта
├── TECHNICAL_SPEC.md            # Техническое задание
├── data/
│   ├── tasks.csv                # Перечень заданий
│   ├── catalog.csv              # Каталог товаров
│   ├── settings.csv             # Настройки по умолчанию
│   └── employee_patterns.csv    # Паттерны начислений по годам
└── examples/
    └── calculation_example.csv  # Пример расчёта
```

## Быстрый старт

1. Загрузите данные из папки `data/`
2. Настройте параметры в интерфейсе
3. Выберите сценарий сотрудника
4. Получите итоговую таблицу с проверками
